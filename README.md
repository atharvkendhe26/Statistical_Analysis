# **📊 Statistical Analysis using Python**

**📖 Project Overview**

Statistical analysis is one of the most important steps in data analytics because it transforms raw numerical data into meaningful business insights. 
This project demonstrates the practical implementation of both Descriptive Statistics and Inferential Statistics using Python in Jupyter Notebook. 
The notebook explores multiple real-world datasets and applies statistical techniques to understand data distribution, detect anomalies, measure variability, 
estimate population parameters, and validate assumptions through hypothesis testing.

The project begins with exploratory analysis of a petrol consumption dataset and gradually introduces advanced statistical concepts such as measures of 
central tendency, skewness, outlier detection, confidence intervals, correlation analysis, and one-sample hypothesis testing. Every concept is implemented 
through Python code and supported with visualizations, making the notebook suitable for learning as well as practical data analysis.

**🎯 Purpose**

The primary purpose of this project is to demonstrate how statistical methods are applied in real-world data analysis using Python. 
Instead of only calculating statistical values, the notebook explains how each statistical technique contributes to understanding data, 
improving decision-making, identifying trends, and drawing valid conclusions from sample observations.

**💼 Business Problem**

Organizations collect large amounts of numerical data every day, but raw data alone cannot support decision-making. 
Businesses require statistical methods to summarize information, detect unusual patterns, estimate future outcomes, 
and validate assumptions before taking action.

This project addresses these challenges by applying descriptive and inferential statistical techniques to multiple datasets. 
The analysis demonstrates how statistics can be used to understand customer behavior, manufacturing performance, operational 
efficiency, and business processes through data-driven decision making.

**📊 Key Insights (Petrol Consumption Dataset)**

* The Petrol Consumption dataset consists of 48 observations and 5 numerical variables, providing a suitable dataset for statistical exploration and analysis.
* The average Petrol Consumption was found to be approximately 576.77, while the median value was 568.50, indicating that the consumption values are relatively
  balanced with only slight variation.
* Paved Highways exhibited the highest variability among all variables, suggesting significant differences in road infrastructure across different regions.
* Skewness analysis showed that some variables are positively skewed, indicating the presence of higher extreme values that influence the overall distribution.
* Boxplot analysis identified outliers in variables such as Petrol Tax and Paved Highways, highlighting observations that differ significantly from the majority
  of the dataset.
* Correlation analysis was performed to examine the relationships between petrol consumption and its influencing factors, including Petrol Tax, Average Income,
  Paved Highways, and Driver License Percentage, helping understand how these variables interact with fuel consumption.
* Histograms, boxplots, and correlation heatmaps provided clear visual insights into the distribution, variability, and relationships among numerical variables,
  making the statistical findings easier to interpret.

**📌 Conclusion**

**1. Confidence Interval (Inferential Statistics)**

After completing the descriptive analysis, the project demonstrates the concept of 95% Confidence Interval (CI) using multiple sample datasets, 
including smartphone screen time, delivery time, and student study hours. The confidence interval was calculated using the sample mean, standard deviation, 
standard error, and critical t-value to estimate the range within which the true population mean is expected to lie. This section illustrates how inferential 
statistics enable analysts to draw reliable conclusions about an entire population based on sample data.

**2. Hypothesis Testing**

The project further applies One-Sample Hypothesis Testing to practical business scenarios such as engine assembly time and baking time. 
For each case, the Null Hypothesis (H₀) and Alternative Hypothesis (H₁) were formulated, followed by the calculation of the test statistic 
and comparison with the critical value. 
Based on the statistical evidence, decisions were made to either accept or reject the null hypothesis. 
This demonstrates how hypothesis testing is used in real-world industries to validate assumptions and support objective, data-driven decision-making.
